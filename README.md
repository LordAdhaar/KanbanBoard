## How To Run This Project Locally 

1. Clone the repo
   ```sh
   git clone git@github.com:LordAdhaar/quicksell-assignment.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   ```
3. Build an app
   ```sh
   npm start
   ```
4. Go to browser and go to 
    ```sh
    http://localhost:3000/
   ```
